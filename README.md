# Retail Sales Data Cleaning and Analysis

## Objective
Clean and analyze a retail sales dataset to identify sales trends and patterns.

## Dataset Source
Kaggle - Superstore Dataset

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Cleaning Techniques

## Data Cleaning Steps
- Standardized column headers
- Checked for duplicate records
- Verified missing values
- Cleaned text fields using TRIM
- Validated numeric and date columns

## Key Insights
- Technology category generated the highest sales
- Sales are concentrated in top-performing states
- Regional differences exist in sales performance
